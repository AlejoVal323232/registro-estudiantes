# Sistema de Gestión de Notas de Estudiantes

Este es un programa desarrollado en C++ que permite registrar a un grupo de estudiantes, capturar sus datos personales, ingresar sus notas y calcular sus promedios automáticamente. Incluye un menú interactivo para consultar la información almacenada de manera organizada.

## ¿Qué hace este programa?

* **Registro de datos:** Permite definir el número de estudiantes a ingresar y guardar sus nombres, IDs y calificaciones.
* **Cálculo automático:** Procesa tres notas por cada estudiante para obtener el promedio final.
* **Menú interactivo:** Ofrece una interfaz sencilla para ver la lista de promedios o verificar el estado de aprobación ("Aprobado" si el promedio es igual o mayor a 3, o "Reprobado" en caso contrario).

## Conceptos que demuestro aquí

* **Estructuras de datos:** Uso de arreglos para almacenar y relacionar nombres, IDs y promedios de los estudiantes.
* **Estructuras de control:** Implementación de ciclos para recorrer y procesar la información de forma eficiente.
* **Menús interactivos:** Uso de ciclos do-while y switch-case para permitir al usuario navegar por las opciones del programa.
* **Lógica condicional:** Aplicación de bloques if-else para evaluar el rendimiento académico de cada alumno.

## Cómo ejecutar este programa

* Asegúrate de tener instalado un compilador de C++ como **Dev-C++**.
* Abre el archivo `registro_estudiantes.cpp` en tu editor.
* Compila y ejecuta el código.
